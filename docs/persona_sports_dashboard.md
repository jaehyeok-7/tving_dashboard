# 스포츠 이용자 이탈 위험 대시보드 (페르소나: Sports 운영)

## 1) Overview: 얼마나 위험한가
목적: 스포츠 이용자 이탈 위험 현황을 빠르게 파악하고, 시즌/비시즌 이상 징후를 조기 탐지

주요 지표
- 스포츠 이용자 이탈 위험군 비율(전체 대비)
- 최근 n주 위험군 추이(시즌/비시즌 구간 표시)
- 경기 종료 후 24시간 / 72시간 재방문율
- 비시즌 유지율(주간 활성/복귀율) 추이
- 위험 급증 Top 경기/팀/리그(최근 1주)

의사결정 포인트
- 이번 주 위험군이 급증했는가
- 어떤 경기/팀에서 위험이 특히 커졌는가
- 비시즌 전환 시점에 유지율이 급락하는가

## 2) 상태 분류: 누가 위험한가
목적: 스포츠 이용자를 유형으로 나누고, 어떤 그룹을 먼저 케어할지 우선순위를 결정

세그먼트 예시
- 팬덤형(특정 팀 중심)
- 라이트형(경기 있을 때만 접속)
- 하이라이트 중심 이용자
- 실시간(라이브) 중심 이용자
- 시즌성 이용자(시즌 종료 후 급감)
- 이탈 위험 이용자(최근 활동 급감/미접속 증가)

제공 가능한 지표
- 세그먼트별 사용자 규모
- 세그먼트별 평균 시청시간 / 시청 빈도
- 경기 종료 후 24h/72h 재방문율(세그먼트별)
- 최근 n일 미접속 비율 / 시청시간 급감 비율
- 비시즌 유지율(세그먼트별)

의사결정 포인트
- 시즌 종료 후 급감 그룹 규모가 커졌는가
- 하이라이트만 보고 떠나는 그룹이 증가했는가
- 복귀 가능성이 높은 위험군부터 우선 개입할지

## 3) 이탈 구간 분석: 어디서 빠지는가
목적: 경기 시청 여정에서 이탈이 급증하는 접점을 찾아, 어떤 부분을 개선해야 하는지 도출

여정 단계(스포츠 흐름)
- 유입 → 경기 진입 → 시청(라이브/하이라이트) → 경기 종료 → 후속 탐색 → 재방문

분석 항목
- 단계별 전환율(유입→경기진입, 경기진입→시청, 종료→후속탐색, 후속탐색→재방문)
- 위험군 vs 유지군의 여정 차이(어느 단계에서 갈리는지)
- 경기 종료 직후 이탈률이 높은 화면/기능(예: 하이라이트 탭, 추천 영역, 알림 설정 유도 등)
- 팀/경기별로 이탈 급증 단계가 다른지 비교

의사결정 포인트
- 경기 종료 직후 후속 탐색 단계에서 이탈이 급증하는가
- 특정 팀/경기에서만 유독 끊기는 단계가 있는가
- 후속 콘텐츠/동선(추천, 하이라이트 묶음) 보강이 필요한가

## 4) 타이밍 분석: 언제 개입해야 하는가
목적: 경기 종료 후 n시간 구간 중 이탈 급증 구간을 찾고, 세그먼트별 최적 개입 타이밍을 도출

제공 지표
- 경기 종료 후 경과시간별 재방문 확률(예: 0~6h, 6~24h, 24~72h)
- 마지막 스포츠 시청 후 경과시간별 이탈 위험도
- 세그먼트별(팬덤형/라이트형/하이라이트형) 최적 개입 시간대
- 비시즌 진입 후 첫 1~2주 구간 위험도 변화

시각화
- 재방문 확률 곡선(경기 종료 후 시간축)
- 세그먼트 × 시간구간 heatmap
- 팀/경기별 위험 피크 타이밍 랭킹

의사결정 포인트
- 어느 시간 구간이 개입 골든타임인가(예: 종료 후 6h, 24h)
- 팬덤형 vs 라이트형의 최적 타이밍이 다른가
- 비시즌에는 개입 주기/타이밍을 어떻게 바꿔야 하는가

## 5) 액션 실험 가이드
액션 유형(실행 옵션)
- 콘텐츠 추천: 경기 종료 직후 유사 경기/하이라이트/관련 콘텐츠 자동 추천
- 알림/푸시: 종료 후 n시간 구간에 재방문 유도 메시지(팀/선수/다음 경기 기반)
- 화면 노출 강화: 스포츠 홈 상단 배너/추천 슬롯, 하이라이트 묶음 고정 노출

제공 지표
- 세그먼트별 추천 액션 우선순위(예: 팬덤형=알림, 라이트형=추천, 하이라이트형=노출강화)
- 액션별 반응률(클릭/진입)과 최종 성과(24h/72h 재방문, 시청시간)

의사결정 포인트
- 어떤 세그먼트에 어떤 액션을 먼저 적용할지
- 푸시 타이밍을 몇 시간대로 설정할지
- 노출 강화(화면 개편)가 필요한지, 메시지/추천만으로 충분한지
